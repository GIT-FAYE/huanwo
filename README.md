# 唤我（HuanWo）

桌面小工具合集，`Ctrl + Space` 随时呼出，零打断工作流。

## 安装

下载 [最新版](https://github.com/GIT-FAYE/huanwo/releases) `HuanWo-Setup-x.x.x.exe`，双击安装。

## 工具（8 个）

| 图标 | 工具 | 分类 | 说明 |
|------|------|------|------|
| 🗜️ | 图片压缩 | 图片 | 质量/大小压缩，支持剪贴板 |
| 🔄 | 图片转格式 | 图片 | JPEG/PNG/WebP/AVIF 互转 |
| ✂️ | 图片裁剪 | 图片 | 自由/固定比例裁剪，拖拽选区 |
| ✏️ | 批量重命名 | 文件 | 序号/替换/前后缀/日期规则 |
| 🌐 | 文本翻译 | 文本 | 千问/DeepSeek/MiMo AI 翻译，20 种语言 |
| 🔍 | 图片识字 | 文本 | 千问/MiMo AI 识别，剪贴板联动 |
| 📱 | 二维码 | 工具 | 文本/URL 生成二维码 |
| 🎨 | 屏幕取色 | 工具 | 全屏截图 + Canvas 取色 + 格式转换 |

## 使用

| 操作 | 效果 |
|------|------|
| `Ctrl + Space` | 呼出工具面板 |
| 输入关键词 | 搜索过滤工具 |
| `Esc` / 点击外部 | 关闭面板 |
| 右键工具卡片 | 固定到顶部 |
| 数字键 `1`-`9` | 快速打开固定工具 |

## 主题

设置页支持 5 种主题：跟随系统 / 浅色 / 深色 / 护眼暖调 / 清新薄荷。

## 设置

| 配置项 | 说明 |
|--------|------|
| 快捷键 | 点按录制组合键 |
| AI 模型 | 千问 / DeepSeek / MiMo 三选一 |
| API Key | 各模型独立配置 |
| 开机自启 | 系统启动时自动运行 |
| 多窗口 | 允许多个工具窗口同时打开 |

## 开发

```bash
npm install
npx electron-vite build
npm run dist    # 打包 NSIS 安装包
```

## 常见问题

**Q: 提示「无法验证发布者」？**  
A: 应用未签名，点击「更多信息 → 仍要运行」即可。

**Q: 快捷键不生效？**  
A: 到设置页修改快捷键，冲突时会提示红色警告。

**Q: 翻译/OCR 报错？**  
A: 检查设置页是否配置了对应 AI 模型的 API Key。

**Q: DeepSeek 能 OCR 吗？**  
A: 不支持，DeepSeek-chat 无视觉能力，请切换到千问或 MiMo。
