# 唤我（HuanWo）

桌面小工具合集，`Ctrl + Space` 随时呼出，零打断工作流。

---

## 安装

下载 `dist\win-unpacked\` 目录，双击 `唤我.exe` 即可运行。

首次运行 Windows 可能提示安全警告，点击「更多信息 → 仍要运行」。

---

## 使用

| 操作 | 效果 |
|------|------|
| `Ctrl + Space` | 呼出/关闭工具面板 |
| `Esc` | 关闭面板 |
| 数字键 `1~9` | 快速打开已固定工具 |
| 右键工具图标 | 固定/取消固定 |
| 分类标签 | 面板顶部按类别筛选工具 |

---

## 工具

### 图片压缩 🗜️
- 拖入图片或点击选择
- 调节质量（1-100）或勾选「限制大小」输入目标 MB
- 支持 JPG / PNG / WebP / AVIF / TIFF / HEIC
- 处理完成后可复制到剪贴板

### 图片转格式 🔄
- 批量转换图片格式
- 支持 WebP / JPEG / PNG / AVIF / TIFF / BMP

### 批量重命名 ✏️
- 序号 / 替换 / 前缀 / 后缀 / 日期 五种规则
- 支持正则替换
- 预览 → 确认 → 执行

### 文本翻译 🌐
- 基于 DeepSeek AI，支持 20 种语言互译
- 翻译偏好：通用 / 跨境电商 / Listing 优化 / 日常口语 / 正式商务 / 技术文档
- 粘贴即译，结果一键复制

### 二维码生成 📱
- 文本 / URL 生成二维码
- 可调尺寸，复制图片或保存 PNG

### 颜色取色 🎨
- HEX / RGB / HSL 三格式
- 24 色预设色板
- 一键复制任意格式

### 设置 ⚙️
- 自定义全局快捷键（点击输入框后按组合键）
- 5 种主题：跟随系统 / 浅色 / 深色 / 护眼 / 清新
- 开机自动启动
- DeepSeek API Key 配置
- 自动更新
- 导出日志

## 常见问题

**Q: 快捷键冲突怎么办？**
设置页可自定义快捷键。如果注册失败，输入框下方会显示红色冲突警告。

**Q: 如何切换主题？**
设置 → 主题 → 选择即可，即时生效。

**Q: 翻译需要什么？**
在设置页填入 DeepSeek API Key（[获取地址](https://platform.deepseek.com/api_keys)）。

**Q: 窗口无法拖动？**
工具窗口支持拖动标题栏。
