# 唤我（HuanWo）

桌面小工具合集，`Alt + Space` 随时呼出，零打断工作流。

---

## 安装

下载 `dist\win-unpacked\` 目录，双击 `唤我.exe` 即可运行。

首次运行 Windows 可能提示安全警告，点击「更多信息 → 仍要运行」。

---

## 使用

| 操作 | 效果 |
|------|------|
| `Alt + Space` | 呼出/关闭工具面板 |
| `Esc` | 关闭面板 |
| 数字键 `1~9` | 快速打开已固定工具 |
| 右键工具图标 | 固定/取消固定 |

---

## 工具

### 图片压缩 🗜️
- 拖入图片或点击选择
- 调节质量（1-100）或勾选「限制大小」输入目标 MB
- 支持 JPG / PNG / WebP / AVIF / TIFF / HEIC
- GIF 仅支持转 WebP 动图
- 处理完成后点 📋 复制到剪贴板

### 图片转格式 🔄
- 批量转换图片格式
- 支持 WebP / JPEG / PNG / AVIF / TIFF / BMP

### 批量重命名 ✏️
- 序号 / 替换 / 前缀 / 后缀 / 日期 五种规则
- 预览 → 确认 → 执行

### 设置 ⚙️
- 自定义全局快捷键（点击输入框后按组合键）
- 开机自动启动
- 导出日志
- 检查更新

---

## 常见问题

**Q: 快捷键冲突怎么办？**
设置页可自定义快捷键，点击输入框后直接按新组合键即可。

**Q: 为什么托盘图标是紫色的？**
品牌色。你说呢？

**Q: 窗口无法拖动？**
工具窗口支持拖动标题栏。
